>> In this LinkedList Implementation, I have created 3 classes
	1. Node
	2. MyLinkedList
	3. MyLinkedListUse

1. Node Class:
    -This class defines the parameter of Node

2. MyLinkedList Class:
    -In this class first taken 3 attributes:
		size:	for maintaining size
		head:	pointer for start element of the list
		tail:	pointer for end elementof the list

    -Then created multiple methods for implementing below functions:
		size()
		toString()
		addFirst(Object data)
		addLast(Object data)
		add(Object data)
		add(int index, Object data)
		set(int index, object data)
